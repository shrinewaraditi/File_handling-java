# File Handling Utility - Java

## 📌 Task
Create a Java program to read, write, and modify text files.

## ⚙️ Features
- Write data into file
- Read data from file
- Modify (append) file content

## 🛠️ Technologies Used
- Java
- File Handling (FileWriter, Scanner)

## ▶️ How to Run
1. Compile:
   javac FileOperations.java

2. Run:
   java FileOperations

## 📂 Files
- FileOperations.java
- sample.txt

## 📖 Output
Program demonstrates:
- File creation
- Reading file content
- Modifying file content
